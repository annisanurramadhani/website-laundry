# Website LaundryAn

Website LaundryAn adalah website layanan laundry modern yang dibuat menggunakan HTML, Bootstrap, dan JavaScript sederhana.

## Fitur
- Landing page menarik (hero section)
- Halaman layanan dengan icon
- Halaman harga dengan detail paket
- Testimoni customer
- Halaman kontak + Google Maps
- Navbar & Footer reusable (component)

## Teknologi
- HTML5
- Bootstrap 5
- JavaScript

## Struktur Project
web-laundry/
├── index.html
├── layanan.html
├── harga.html
├── testimoni.html
├── kontak.html
├── components/
│ ├── navbar.html
│ └── footer.html

## Live Demo
https://laundryan.vercel.app/

## Tujuan Project
Project ini dibuat untuk latihan membuat website bisnis sederhana dengan tampilan modern dan struktur yang rapi.
